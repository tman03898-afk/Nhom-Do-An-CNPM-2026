# Hệ thống Quản lý Phòng Trọ — Backend (nhánh `be-dev`)

Nhánh này chỉ chứa **`phong-tro-backend`**. Toàn bộ mã nguồn gồm cả React frontend nằm ở nhánh **`main`**.

---

## Thư mục

| Thư mục | Mô tả |
|---------|--------|
| `phong-tro-backend/` | API Express, routes, cấu hình |

---

## Chạy backend

```bash
cd phong-tro-backend
npm install
cp .env.example .env   # chỉnh biến môi trường
npm run dev
```

Chi tiết xem `phong-tro-backend/README.md` (nếu có).
